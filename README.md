#Alarm Clock

##Description
This is a simple alarm clock app built utilizing Node JS, Gulp, JS/JQuery and Moment JS. It has a time-set option and snooze option.

##Build Process

* Git clone repo
* Open your CLI
* Install node JS
* run npm install
* run bower install
* run gulp build
* run gulp serve
